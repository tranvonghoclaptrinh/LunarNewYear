## 📖 Giới Thiệu

**Website Tết Đồng Nai 2026** là một website tĩnh được thiết kế nhằm giới thiệu những nét đẹp đặc trưng của tỉnh Đồng Nai trong dịp Tết cổ truyền Việt Nam, bao gồm:

- 🍜 Ẩm thực đặc sản địa phương  
- 🏔️ Địa điểm du lịch nổi bật  
- 🎭 Văn hóa, lễ hội truyền thống  
- 📖 Giá trị truyền thống và cộng đồng  

Website tập trung vào **UI hiện đại, hiệu ứng mượt, hỗ trợ Dark Mode**, và trải nghiệm người dùng tốt trên mọi thiết bị.

## 🌙 Dark Mode

Tự động lưu trạng thái:

```js
localStorage.setItem("darkMode", true)
```
```css
body.dark-mode {
    background: #0f172a;
    color: white;
}
```
